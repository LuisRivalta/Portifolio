
## 🚀 Começando com o Projeto

Este é um projeto [Next.js](https://nextjs.org) inicializado através do comando [`create-next-app`](https://www.google.com/search?q=%5Bhttps://nextjs.org/docs/app/api-reference/cli/create-next-app%5D(https://nextjs.org/docs/app/api-reference/cli/create-next-app)). Ele já vem configurado com as melhores práticas de performance e estrutura de pastas.

### 🛠️ Como rodar localmente

Para colocar o ambiente de desenvolvimento de pé, execute um dos comandos abaixo no seu terminal:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev

```

Após iniciar o servidor, acesse **[http://localhost:3000](https://www.google.com/search?q=http://localhost:3000)** no seu navegador para visualizar a aplicação.

### 📝 Editando o código

O ponto de entrada principal é o arquivo `app/page.tsx`. Graças ao *Fast Refresh*, qualquer alteração que você fizer e salvar será refletida instantaneamente no navegador, sem que você precise atualizar a página manualmente.

### 🎨 Estilização e Fontes

O projeto utiliza o pacote [`next/font`](https://www.google.com/search?q=%5Bhttps://nextjs.org/docs/app/building-your-application/optimizing/fonts%5D(https://nextjs.org/docs/app/building-your-application/optimizing/fonts)) para carregar e otimizar automaticamente a **Geist**, a nova família de fontes da Vercel, garantindo uma ótima experiência de leitura e performance (CLS zero).
